Case vahendid

CASE-vahendid aitavad automatiseerida tegevusi süsteemiarenduse
elutsükli erinevates faasides. Näiteks prototüüpimise korral on
olemas spetsiaalsed tarkvaralised vahendid, millega mugavalt ja
kiirelt luua rakenduse ekraanipiltide graafilisi mudeleid.

CASE Vahendeid kasutatakse:

  1. analüüsimiseks
  2. protsessi dokummenteerimiseks
  3. erinevtas faasides tegevuste automatiseerimiseks

CASE vahendid jagunevad ka kaheks:

  1. Lowercase vahenditeks
  2. Uppercase vahenditeks

Arendusmudeli elutsüklis saab kasutada vahendeid nagu:

  1. Data Modelling
  2. Analysis
  3. User Interface Prototyping
  4. Code Generator

Ma olen kasutanud:

  1. Powerpoint
  2. Github
